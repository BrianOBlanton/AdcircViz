Basic Implementation of AdcircViz

AdcircViz relies on making connections to the OPeNDAP server using the MATLAB toolbox nctoolbox. This toolbox (http://nctoolbox.github.com/nctoolbox) is used extensively to open netCDF data files, without accessing the data components until they are needed. Each graphic is generated using MATLAB functions and tools “on the fly”; the color map, color limits and color increment are specified by the user with default values appropriate to the scalar field being rendered. The default colormap is one similar to the NOAA colormap currently used for NHC surge maps: [Image of noaa_cmap].

All needed MATLAB code is included in the application’s distribution package. Other than having MATLAB, no other MATLAB or external dependencies exist.

Running AdcircViz

Start MATLAB on your computer. In MATLAB, navigate the Current Folder to adcircviz\trunk, where AdcircViz.m and a few other m-files are stored. The current path will be referred to as the HOME path in this manual. To open the GUI window of the AdcircViz Tool, type AdcircViz in the Command Window to get a default view. Note: Only one instance of AdcircViz GUI is allowed to run at a time. Close the previous one before open another.

To change the fontsize of the GUI window, use the FontOffset parameter to adjust, e.g., type

close all; AdcircViz('FontOffset',-2)

in the Command Window to get a new GUI window with fontsize reduced by 2.

To change the units of the map (default “Meters” ) to “Feet”, use the “Units” parameter to adjust, e.g., type

close all; AdcircViz(‘Units’,’Feet’).

To change other settings of the map, see Section 5: Changing the Catalog Parameters.
